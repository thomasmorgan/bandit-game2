Instructions

Every trial takes place in a specific country, for example, Portugal or Ethiopia.

You might return to a country you have already been to and the treasure will be in the same location as before. So you should try to remember where you found it last time.

You don't have to check locations at all, so if you can remember where the treasure is, or if you just feel like guessing, you can press the "Ready to Decide" button at any time and then make your final decision
